
Backups a directory.

Syntax:
   aBackup now [{directory to backup}] [{repository}]

Parameters:
   directory to backup: Optional directory to be backed up. If not specified, 
                        current directory will be backed up.
   repository:          Directory to store the backup file. It isn't necessary
                        if {directory to backup} has been previously configured
						to be backed up by calling "aBackup init" (run 
						"aBackup help init" for details).
