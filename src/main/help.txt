
aBackup.bat - v0.1.0 - A quick and dirty backup tool for Windows.

Syntax:
   aBackup [help] {command}

Commands:
   help:    Displays help about a given command.
   now:     Makes a backup of a directory and saves it in another directory.
   init:    Prepares a directory for repetitive backups.
   restore: Restores a previously made backup.
